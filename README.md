# Mission Center - 任务管理中心

一个现代化的任务管理应用，支持创建、编辑、跟踪任务，并提供多种分类和状态管理功能。

## 技术栈

- React 18
- TypeScript
- Vite
- React Router
- Zustand (状态管理)

## 功能特性

- ✅ 任务创建、编辑、删除
- ✅ 任务状态管理（待办、进行中、审核中、已完成、已取消）
- ✅ 任务优先级管理（低、中、高、紧急）
- ✅ 任务分类（工作、个人、学习、健康、财务、其他）
- ✅ 任务标签系统
- ✅ 任务搜索和筛选

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
